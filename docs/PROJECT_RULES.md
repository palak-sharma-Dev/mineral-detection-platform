# Development Rules

## General

- Never redesign completed frontend pages.
- Preserve enterprise design.
- Keep responsive.
- Use reusable components.

## UI Rules

- Professional
- Scientific
- Minimal

Do NOT use:

- Neon
- Crypto UI
- Fake charts
- Fake analytics

## Coding Rules

- TypeScript only
- Modular components
- No duplicate code

## Backend Rules

- Follow Deployment API.
- Use JWT.
- Role-based access.

## AI Rules

- Async jobs only.
- Poll status.
- Don't block UI.
